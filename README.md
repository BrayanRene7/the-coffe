# ☕ The Coffee - Sitio Web

## 📌 Descripción

Este proyecto consiste en el desarrollo de una página web para una cafetería llamada **The Coffee**, donde los usuarios pueden conocer los productos, información del negocio, ubicaciones de sucursales y acceder a opciones de registro e inicio de sesión.

## 🎯 Objetivo

El objetivo del proyecto es crear una interfaz web atractiva y funcional que permita mostrar los servicios y productos de una cafetería, mejorando la experiencia del usuario.

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## 📂 Estructura del proyecto

* `index.html` → Página principal
* `productos.html` → Catálogo de productos
* `nosotros.html` → Información sobre la cafetería
* `sucursales.html` → Ubicación de sucursales
* `login.html` → Inicio de sesión
* `registro.html` → Registro de usuarios
* `CSS/` → Archivos de estilos
* `js/` → Scripts JavaScript
* `IMG/` → Imágenes del proyecto

## ▶️ Cómo usar el proyecto

1. Clonar o descargar el repositorio
2. Abrir el archivo `index.html` en un navegador web
3. Navegar por las diferentes secciones del sitio
